# 任务二：地区经济规模对比 — 设计说明

---

## 一、总体概述

本任务旨在通过10个静态图表，从不同维度展示中国各省份/地区之间的GDP规模差异，包括总量排名、产业构成、时间对比、空间分布、趋势演变、排名变迁、份额结构、增量贡献等多角度分析。

**数据来源：**
- `2015-2024年各省份GDP统计表.csv` — 31个省份10年GDP及三大产业数据
- `2015和2024年各省GDP产业数据.csv` — 31个省份2015/2024年细分行业数据
- `china_provinces.geojson` — 中国省份行政区划矢量数据（阿里云DataV）

**技术方案：** Python 3 + Matplotlib + GeoPandas + Pandas，输出300dpi高清PNG图片

### 图表5：基于变形地图的各省/地区GDP发展曲线可视化

**文件：** `图表5_变形地图GDP发展曲线.png`

**可视化目标：** 以 Dorling Cartogram 圆形气泡形式近似中国地图布局，气泡面积编码2024年GDP规模，圆内嵌入2015-2024年发展曲线，同时传递"空间分布"与"时间趋势"两个维度的信息。

**图表类型：** Dorling Cartogram（圆形变形地图）+ 圆内 Sparkline（迷你折线图）

**设计要点：**
- 31个省份各对应一个圆形气泡，位置以真实经纬度为初始锚点，近似中国地图布局
- 气泡**面积**（而非半径）与2024年GDP成正比，采用平方根映射：`radius ∝ √GDP`，半径范围 1.0–4.0（数据单位）
- 气泡**填充色**使用 YlGnBu 色带，浅黄→青绿→深蓝，GDP越高颜色越深
- 省份名称显示在圆内上半部，白色字体加深海军蓝描边（`path_effects`），在深浅两种底色上均清晰可读
- **发展曲线**绘制在圆内下半部，覆盖圆心下方约60%高度区域：
  - 珊瑚红主线（`#ff6b6b`）+ 深红描边提升对比
  - 浅粉色圆点标记2015年起点，深红色圆点标记2024年终点
  - 曲线纵向归一化到各省自身 min-max 区间，突出趋势形态而非绝对量
- 使用**力导向排斥算法**解决圆形重叠：以地理坐标为初始位置，迭代推开相互重叠的圆，直到所有圆间隙 ≥ 0.8 单位，保证全部31省可见且无遮挡

**核心代码：**
```python
# 气泡半径：平方根映射，面积正比于GDP
ratio = np.sqrt((gdp_val - gdp_min) / (gdp_max - gdp_min))
radius = min_radius + ratio * (max_radius - min_radius)

# 力导向排斥算法消除重叠
resolved_pos = {k: np.array(v, dtype=float) for k, v in map_positions.items()}
for _ in range(3000):
    for i in range(len(keys)):
        for j in range(i + 1, len(keys)):
            diff = resolved_pos[k2] - resolved_pos[k1]
            dist = np.linalg.norm(diff)
            min_dist = radii_dict[k1] + radii_dict[k2] + gap
            if dist < min_dist:
                push = (min_dist - dist) / 2 * diff / dist
                resolved_pos[k1] -= push
                resolved_pos[k2] += push

# 圆内发展曲线（归一化到自身 min-max）
curve_cy = lat - radius * 0.30       # 曲线区域中心偏下
x_pts = np.linspace(lon - radius*0.8, lon + radius*0.8, n_pts)
y_pts = curve_cy - radius*0.275 + norm_vals * radius*0.55
ax.plot(x_pts, y_pts, color='#ff6b6b', linewidth=lw)
```

**图表解读：**
- 广东、江苏圆形最大且颜色最深，是2024年GDP规模最突出的省份
- 珊瑚红发展曲线在所有省份均呈上升形态，但斜率差异明显：贵州、云南等西部省份曲线更陡，增速更快
- 东部沿海省份气泡密集且体量大，西部省份气泡小而分散，空间梯度一目了然
- 力导向算法保留了原始地理方位关系（东西南北相对位置），同时确保信息可读性

## 三、图表间逻辑关系

```
图表1（总量排名）── 谁大谁小？
    │
    ├── 图表2（产业构成）── 大的省份靠什么产业支撑？
    ├── 图表3（时间对比）── 十年间谁增长最快？
    ├── 图表4（地图分布）── GDP在地理上如何分布？
    ├── 图表5（趋势+空间）── 各省增长趋势在空间上如何呈现？
    ├── 图表6（排名变迁）── 十年间排名如何升降？
    ├── 图表7（规模玫瑰）── 各省规模与地区归属如何分布？
    ├── 图表8（份额演变）── 各省占全国比重如何变化？
    ├── 图表9（增量贡献）── 谁对全国增长贡献最大？
    └── 图表10（总量×人均）── 总量大是否等于人均富？
```

十个图表从**排名→结构→时间→空间→趋势→变迁→规模→份额→增量→双维对比**十个维度，全面刻画了中国地区经济规模的差异格局。

## 四、关键技术说明

### 省份名称匹配

CSV数据使用短名（如"广东"），GeoJSON使用全称（如"广东省"），通过映射表实现匹配：

```python
PROVINCE_NAME_MAP = {
    '北京': '北京市', '广东': '广东省', '内蒙古': '内蒙古自治区', ...
}
```

### Dorling Cartogram 圆形布局

以各省真实经纬度为初始锚点（近似地图方位），运行力导向排斥算法推开重叠圆，最终布局在保留地理方位关系的同时保证所有气泡清晰可见：

```
西北：新疆 甘肃 青海 宁夏 陕西
东北：黑龙江 吉林 辽宁 内蒙古
华北：北京 天津 河北 山西
华中/华东：河南 湖北 湖南 山东 安徽 江苏 上海 浙江 江西 福建
西南：西藏 四川 重庆 云南 贵州
华南：广东 广西 海南
```
