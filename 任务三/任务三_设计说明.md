# 任务三：GDP相关分析 — 设计说明

## 一、总体概述

本任务旨在通过8个静态图表，从多维度分析中国各省份GDP发展与产业结构之间的相关性，包括构成关系、相关性散点、相关性热力图、多维平行坐标、桑基图、PCA双标图及3D可视化。

**数据来源：**
- `2015-2024年各省份GDP统计表.csv` — 31个省份10年GDP及三大产业数据
- `2015和2024年各省GDP产业数据.csv` — 31个省份2015/2024年细分行业数据

**技术方案：** Python 3 + Matplotlib + Pandas + Scikit-learn (PCA)，输出300dpi高清PNG图片

**地区划分（四大地区，参照国家统计局标准）：**
- 东部：北京、天津、河北、上海、江苏、浙江、福建、山东、广东、海南
- 中部：山西、安徽、江西、河南、湖北、湖南
- 西部：内蒙古、广西、重庆、四川、贵州、云南、西藏、陕西、甘肃、青海、宁夏、新疆
- 东北：辽宁、吉林、黑龙江

### 图表1：2024年各省GDP与全国GDP构成关系（旭日图）

**文件：** `图表1_GDP构成旭日图.png`

**可视化目标：** 展示全国GDP从地区→省份→产业的三层构成关系。

**图表类型：** 三层旭日图

**设计要点：**
- 内环：四大地区（东部、中部、西部、东北）
- 中环：各省份（按地区归属分组）
- 外环：三大产业（第一/二/三产业，同地区色系浅→中→深）
- 中心标注全国GDP总量
- 图例：地区与产业合并为一排图例置于图表正下方

**配色方案：**
- 东部：红色 `#e74c3c`
- 中部：绿色 `#2ecc71`
- 西部：紫色 `#9b59b6`
- 东北：蓝色 `#3498db`
- 外环产业：各地区对应色系，浅色=第一产业，中色=第二产业，深色=第三产业

**核心代码：**
```python
# 地区分组
region_order = ['东部', '中部', '西部', '东北']
region_gdp = df_2024.groupby('地区')['地区生产总值 (GDP)_亿元'].sum()

# 内环：地区
for region in region_order:

    span = gdp / total_gdp * 360
    draw_wedge(ax, t1, t2, r_inner, r_mid1, REGION_COLORS[region])

          bbox_to_anchor=(0.5, -0.02), ncol=7,
          title='内环：地区          外环：产业深浅')
```

**图表解读：**
- 东部地区GDP占全国比重最大（约50%）
- 广东、江苏、山东、浙江四省GDP合计超过全国1/3
- 北京、上海第三产业占比超过75%，结构差异明显

### 图表6：省份发展模式聚类（PCA双标图）

**文件：** `图表6_PCA双标图.png`

**可视化目标：** 使用主成分分析揭示省份发展模式，识别不同类型的经济发展路径。

**图表类型：** PCA双标图（Biplot）

**设计要点：**
- 散点：各省份（按四大地区着色）
- 红色箭头：原始变量载荷方向（GDP总量、增速、三产业占比、人均GDP）
- PC1+PC2 合计解释方差标注于轴标签

**核心代码：**
```python
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA

X = StandardScaler().fit_transform(features.values)
pca = PCA(n_components=2)
coords = pca.fit_transform(X)
loadings = pca.components_.T  # shape (n_features, 2)
```

**图表解读：**
- PC1主要反映经济规模（GDP总量、人均GDP）
- PC2主要反映产业结构转型程度
- 省份在图中的位置揭示其发展模式特征

## 五、关键技术说明

### 四大地区划分

```python
EAST_CHINA    = ['北京', '天津', '河北', '上海', '江苏', '浙江', '福建', '山东', '广东', '海南']
CENTRAL_CHINA = ['山西', '安徽', '江西', '河南', '湖北', '湖南']
WEST_CHINA    = ['内蒙古', '广西', '重庆', '四川', '贵州', '云南', '西藏', '陕西', '甘肃', '青海', '宁夏', '新疆']
NORTHEAST     = ['辽宁', '吉林', '黑龙江']
```

### 旭日图绘制

使用 `matplotlib.patches.Wedge` 绘制扇形，通过角度和半径控制各层大小。图例采用地区与产业合并方式，置于图表正下方，产业色块使用东部红系浅/中/深三档示意外环颜色编码逻辑。

### PCA降维

使用 scikit-learn 的 `StandardScaler` 和 `PCA` 进行标准化和降维，保留2个主成分进行可视化。
